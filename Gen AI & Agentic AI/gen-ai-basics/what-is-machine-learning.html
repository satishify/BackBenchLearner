What is Machine Learning (ML)

Machine Learning is a subset of AI where models learn patterns from data instead of relying only on hardcoded rules.

Core idea

A system improves performance on a task as it sees more experience/data.

Main learning types

TypeDataExamples
SupervisedLabeled input-output pairsSpam detection, price prediction
UnsupervisedNo labelsClustering, anomaly patterns
flowchart LR A[Data] --> B{Labels?} B -- Yes --> C[Supervised training] B -- No --> D[Unsupervised learning] C --> E[Prediction] D --> F[Pattern discovery]

Practical workflow

  1. Collect and clean data.
  2. Split into train/test sets.
  3. Train a model.
  4. Evaluate with proper metrics.
  5. Deploy and monitor.
